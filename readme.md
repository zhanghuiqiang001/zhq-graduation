# zhq-graduation仓库的说明

#### 摘要:

随着微信的普及,微信小程序的流行,以及网上购物的盛行,基于微信小程序我设计了一个电商小程序.微信小程序是一种不需要安装即可打开使用的应用,即:即用即走.用户只需要通过进入本系统的微信小程序,就可以实现网上购物(由于无法模拟支付,故没有支付系统)





#### 功能模块:

- 微信授权登录
- 浏览商品
- 分类列表
- 商品详情
- 分享,客服
- 加入购物车
- 获取位置等信息
- 支付订单
- 反馈意见
